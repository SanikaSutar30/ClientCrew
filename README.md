# ClientCrew 🚀

ClientCrew is a full-stack web application that combines CRM, Project Management, and Team Communication in one platform.

## Tech Stack
- Frontend: React (Vite), Tailwind CSS
- Backend: Spring Boot, Hibernate
- Database: MySQL

## Features
- Customer Management
- Project & Task Tracking
- Team Collaboration (Messaging)
- Role-Based Access (Admin, Manager, Employee, Customer)
- Dashboard & Reports
